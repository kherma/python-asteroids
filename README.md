# Python Asteroids

`python-asteroids` is a small arcade-style Asteroids clone built with Python and Pygame. You control a triangular ship, dodge incoming asteroids, and shoot them to break them apart. The game ends when an asteroid collides with the player.

## Requirements

- Python 3.13+
- `uv`

## How to run

1. Install dependencies:

```bash
uv sync
```

2. Start the game:

```bash
uv run main.py
```

## Controls

- `W`: move forward
- `S`: move backward
- `A`: rotate left
- `D`: rotate right
- `Space`: shoot
